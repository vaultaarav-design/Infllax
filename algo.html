🎯 PAGE OVERVIEW — ISI ALGO ENGINE

ISI Algo Engine ek institutional-grade trading dashboard hai. Chart + Real-time signals + Multi-account order routing ek hi jagah. Python algo engine Firebase ke through connect hota hai aur MT5 / Custom API pe orders execute karta hai.

📊 CHART — KAISE USE KAREIN

Symbol Top bar mein dropdown se symbol badlo — XAUUSD, EURUSD, NAS100, BTCUSD etc.
Timeframe M1 · M5 · M15 · M30 · H1 · H2 · H4 · D1 · W1 · MN — click karke switch karo.
Data Real Yahoo Finance data load hota hai. Agar CORS block ho toh realistic simulation fallback.
Live Tick Har 2 second mein price update hoti hai — last candle live dikhta hai.

📐 DRAWING TOOLS — CHART PE DRAW KARO

─ H-Line Click karo — horizontal level draw. Support/Resistance mark karo.
│ V-Line Click karo — vertical time marker. Event/session boundary mark karo.
╱ Trend Click start point → click end point — trendline draw hogi.
▭ Rect Click drag — rectangle/zone draw. Order block ya range mark karo.
φ Fib Click high → click low — Fibonacci levels auto-draw: 0.236, 0.382, 0.5, 0.618, 0.786.
T Text Click karo → text type karo — chart pe label lagao.
🗑 Clear Saari drawings ek baar mein hata do.
✕ None Drawing mode off — normal chart interact karo.

🔌 CONNECT TAB — MULTI ACCOUNT

Settings mein banaye clusters/accounts yahan card form mein dikhenge. ⚡ SET ACTIVE dabao — us account pe orders jaayenge. 📊 POSITIONS se live balance/equity dekho.

🧠 ICT / SMC LAYERS — CHART PE KYA DIKHTA HAI

📍 SIG Algo signals ka entry/SL/TP chart pe lines ke form mein.
📦 OB Order Blocks — bullish (green) / bearish (red) zones jahan institutions ne position liya.
◈ FVG Fair Value Gaps — imbalance zones. Price yahan wapas aata hai fill karne.
💧 LIQ Liquidity levels — swing highs/lows jahan stops hain. Institutions yahan sweep karte hain.
⚡ BOS Break of Structure + CHoCH — trend change ya continuation confirm karta hai.
〜 EW Elliott Wave — 5-wave impulse + 3-wave correction auto-detection. Wave labels chart pe.

📋 ORDER FLOW

1. Signal aaya → SIGNALS tab mein card dikhega.
2. ✅ Approve karo → ORDER tab mein auto-fill.
3. Account select karo → ▲ EXECUTE dabao.
4. Firebase → Python engine → MT5/API execute.
5. 📋 Order Tracker (bottom-right) mein status dekho.
